Using Bullets and Numbers

Now is an opportune time to show you how to create bullets. To create bullets in your Resume following the instructions below.

  • Click on Format from the tool bar. And, from the drop down menu select bullets and numbering.

  • Next, you will see the screen below. Click the types box and select bullets.

  • If you would like to change styles, select the styles box and choose one.
  • Choose apply and close. You have just created bullets in your document.

To use numbers instead of bullets do the following.

  • Select Format from the menu bar.
  • Select numbers from the type box.
  • Change the style of numbers by clicking on the style box and changing the style.
  • Click "apply" and then "ok"
