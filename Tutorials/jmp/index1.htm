Welcome to the AbiWord Help!

Abiword is a free word processing program similar to Microsoft Word, provided by SourceGear Corporation. In AbiWord Help, you'll learn how to use AbiWord to type specific kinds of documents. Our intention is to show you, the user, how to learn the main functions of AbiWord through task-oriented tutorials.

We've arranged the tutorials into three main types of tasks: a letter, a resume, and a party invitation. Below you will find an overview of what functions you will learn in each tutorial.

  • Tutorial 1, you will be writing a letter which focuses on the fundamentals of using AbiWord. In this tutorial you'll learn basic functions such as how to: create a new document, open and close a document, save and print a document, word count, insert the date and time, number pages, and check for spelling errors.

  • Tutorial 2, you will be writing a resume which focuses on key functions such as: copying and pasting text, find and replace words, changing font style using the tool bar, find and replace text, and inserting bullets and numbering.

  • Tutorial 3, you will be writing a party invitation which focuses on key functions such as: pictures, copying and pasting text, using margins, changing font color and style, allignment and justification, inserting a graphic and clip art, undo and zoom.

Note: The index is a list of AbiWord functions not associated with the three tutuorials listed above. Select a link in the index to display the definition and use of each function.

Now that you know about each tutorial and the index, let's look at how to navigate AbiWord Help.

The navigation of AbiWord Help is simple. On the left side of the screen you will find navigation links to each of the three tutorials, as well as the function index. At any time, you can choose these links to view a tutorial or a specific function. Now that you are familiar with AbiWord Help, let's get started on tutorial 1 ,"Writing A Letter".
