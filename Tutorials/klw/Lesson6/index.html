Overview

At this point in the tutorial, you should feel comfortable with some of the more basic and intermediate features and functions of AbiWord. In this section of the tutorial you will learn to apply some of the more advanced features of the system.

Objectives

By the time you complete this section in the tutorial, you should know how to perform the following functions in AbiWord:

  • Apply subscript/superscript
  • Create a bookmark
  • Create a hyperlink

If you'd like to move forward and learn how to apply subscript/superscript to text, select Next. If you'd like to go to a different section within this lesson or to a new lesson, choose from one of the options in the Navigation bar.
