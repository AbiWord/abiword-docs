Close a Document

At this point you know how to start AbiWord, create and save your document. Now let's close the document you just created by following these few easy steps:

  • Select File located on the Menu bar
  • Select Close

    This will close your document and a new blank document will appear.

AbiWord offers your two other options for closing a document:

  1. Select the Close button located in the top right corner of the AbiWord window.

    This closes AbiWord and all documents open within it. You will be prompted to save your changes if there were any changes made to your document since the last time you saved it.

  2. Use the keyboard shortcut to close your document. Just press Ctrl+w (the Ctrl key and the "w" key at the same time).

You have just completed this section of Lesson 1 and can now close a document. If you'd like to learn how to open a document, select Next. If you'd like to go to a different section within this lesson or to a new lesson, choose from one of the options in the Navigation bar.
