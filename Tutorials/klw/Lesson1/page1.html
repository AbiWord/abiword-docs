Start AbiWord

If you already have AbiWord installed on your computer, open AbiWord by following these four easy steps:

  • Open the Start menu by selecting the Start button
  • Select Programs
  • Select AbiWord
  • Select AbiWord

Note: You can also start AbiWord by selecting the AbiWord icon on your desktop.

You have just completed this section of Lesson 1 and can now start AbiWord. If you'd like to learn how to create a new document, select Next. If you'd like to go to a different section within this lesson or to a new lesson, choose from one of the options in the Navigation bar.

Create a New Document

When you start AbiWord, a new document automatically launches. However, if you need to create any additional documents from within AbiWord, do the following:

  • Select the Create a new document button

Note: You can also create a new document by selecting File from the Menu bar and then selecting New.

You have just completed this section of Lesson 1 and can now create a new document. If you'd like to learn how to save a document, select Next. If you'd like to go to a different section within this lesson or to a new lesson, choose from one of the options in the Navigation bar.
