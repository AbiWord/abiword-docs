Print a Document

Printing your document from within AbiWord is easy, and doesn't differ from any other program you may use.

Let's print the document you created earlier by following these five easy steps:

  • Open the document you created earlier in this lesson
  • Select the Print the document button

    The Print dialog opens. You can change the options here to suit your needs.

  • Select your printer

    Note: The printer shown will be your computer's 'default' printer. You can change your printer's properties by selecting the Properties button. Since this is outside the scope of this tutorial, we won't cover Printer Properties.

  • Set the number of copies to print

    Note: You can change the number of copies by entering the number directly, or you can use the up and down arrows to select the number of copies. For your document, let's just leave it set to 1.

  • Choose which pages to print

    Note: You can select certain pages to print if your document is more than one page long. Just select the radio button next to the Pages option in the Print Range section, then enter the range of pages to print. Let's just print all pages right now, so leave All checked.

  • Select OK

Congratulations! You have just completed the first section of the AbiWord Tutorial and can now start AbiWord; create, save, close, open and print a document; as well as check your document for spelling errors. If you'd like to learn how to format your document, select Next. If you'd like to go to a different section within this lesson or to a new lesson, choose from one of the options in the Navigation bar.
