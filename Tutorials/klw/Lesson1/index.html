Overview

This first section of the tutorial will introduce you to some basic concepts of word processing using AbiWord. You will learn how to use AbiWord to create a simple document, such as a note or informal letter, as well as how to create more advanced documents while progressively utilizing advanced features of the system.

Objectives

By the time you've completed this section of the tutorial, you should know how to perform the following functions in AbiWord:

  • Start AbiWord
  • Create a new document
  • Save a document
  • Close a document
  • Open a document
  • Spellcheck a document
  • Print a document

If you'd like to move forward and learn how to start AbiWord, select Next. If you'd like to go to a different section within this lesson or to a new lesson, choose from one of the options in the Navigation bar.
