Undo/Redo Actions

AbiWord allows you to return to a previous state by undoing the effects of one or more commands as well as redoing an action. The undo command is a valuable feature that lets you try unknown commands with less risk, because you can always return to the previous state. Also, if you accidentally press the wrong function key, you can undo your mistake. The redo command reapplies actions you may have undone.

Undo

Let's undo an action by following these few steps:

  • Select some text in your document
  • Select the Cut button

    The text you selected disappears.

  • Select the Undo editing button

    The text you cut reappears.

Redo

Now, let's assume you actually do want that text cut from your document. To redo your last action, follow these simple instructions:

  • Select the Redo editing button
