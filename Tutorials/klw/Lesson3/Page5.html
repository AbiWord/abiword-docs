Page Breaks

A page break is the place where a page ends and a new page begins. A word processor will automatically insert a page break after a certain number of lines; or the you can enter a command to put a page break in a particular place. If you find that you need to skip ahead and create another blank page to an existing document, this is a wonderful short cut.

Let's insert a page break in your document by following these four easy steps:

  • Open the document you created in Lesson 1
  • Place cursor at the end of text
  • Select Insert located on the Menu bar
  • Select Break

    This will create a break and move the cursor to the next page.

You have just completed this section of Lesson 3 and can now insert a page break into your documents. If you would like to learn how to find and replace words, select Next.
