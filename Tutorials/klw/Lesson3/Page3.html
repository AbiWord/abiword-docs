Setup Margins

Use margins to determine how far your text will go across the page. This option will place stopping points for the text on the top, bottom, left, and right sides of the page. This makes the document look clean and aligned.

  • Select Format from the Menu bar
  • Select Document

    The Tabs dialog box appears containing two tabs, Page and Margin.

  • Select the Margin tab
  • Determine where you would like the margins to be set
  • Determine the dimensions for the top margin, bottom margin, left margin, and right margin

You have just completed this section of Lesson 3 and can now set the margins in your documents. If you would like to learn how to set your tabs, select Next. If you'd like to go to a different section within this lesson or to a new lesson, choose from one of the options in the Navigation bar.
