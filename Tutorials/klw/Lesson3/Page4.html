Format Tabs

A tab is a character which when displayed or printed moves the cursor, and any text in front of it, to the next tab stop. Setting tabs is useful for indenting and lining up columns.

Let's set some tab stops in your document by following these few basic steps:

  • Open the document you created in Lesson 1
  • Select Format located on the Menu bar
  • Select Tabs
  • Set tab stops that best serve your needs

You have just completed this section of Lesson 3 and can format tabs in your documents. If you would like to learn how to insert a page break, select Next. If you'd like to go to a different section within this lesson or to a new lesson, choose from one of the options in the Navigation bar.
