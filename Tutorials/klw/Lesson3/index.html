Overview

Lessons 1 and 2 provided you with instructions on using some of the basic features of AbiWord. This lesson will teach you more intermediate skills involving editing documents. Editing documents is essential to document production.

Objectives

At the end of this section of the tutorial, you should know how to perform the following functions of AbiWord:

  • Page setup
  • Format font
  • Format margins
  • Format tabs
  • Insert page breaks
  • Find word(s)
  • Replace word(s)

If you'd like to move forward and learn how to set the overall shape and size of the pages in your document, select Next. If you'd like to go to a different section within this lesson or to a new lesson, chose from one of the options in the Navigation bar.
