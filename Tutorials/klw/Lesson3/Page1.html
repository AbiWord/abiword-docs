Page Setup

The Page Setup feature in AbiWord allows you to determine the size and orientation of the pages in your document.

Paper Size and Orientation

AbiWord provides you with many pre-set paper sizes, including most of those in normal use around the world. For most documents you create, you will be using the standard letter size (8.5 x 11).

Let's select your document's paper size by following these few basic instructions:

  • Open the document you created in Lesson 1
  • Select Page Setup from the File menu located on the Menu bar

    The Page Setup dialog appears.

  • Select the paper from the Paper Size dialog
  • Select Width
  • Select Height
  • Select Units
  • Select the Orientation

    Note: Typically, you will use the Portrait orientation. A page with portrait orientation, typical for letters, memos, and other text documents, is taller than it is wide. Whereas, a page with landscape orientation is wider than it is tall.

  • Select Ok

You have just completed this section of Lesson 3 and can now set the page orientation of your documents. If you would like to learn how to format the font of your document by changing the font type and size, select Next.
