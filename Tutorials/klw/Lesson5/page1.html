Apply a Style

When you start AbiWord the new blank document, based on the Normal template, uses the Normal style for the text that you type. This means that when you start typing, AbiWord uses the font, font size, line spacing, text alignment, and other formats currently defined for the Normal style. Based on the Normal style, the base style for the Normal template, you can use this style as a building block for other styles in the template.

Choose from a number of other styles besides Normal available in the Normal template. You can preview these styles from the Styles dialog box.

Let's apply a different style to some text in your document by following these three easy steps:

  • Place your cursor in the paragraph where you want to apply the new style
  • Select the Style drop-down arrow located on the Formatting toolbar
  • Select a style from the list

You have just completed this section of Lesson 5 and can now apply a new style to some of the text in your documents. If you'd like to learn how to apply a style from a larger selection of styles located in the Style dialog box, select Next.
