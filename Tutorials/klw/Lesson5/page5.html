Delete a Style

AbiWord allows you to delete a style that you have currently applied to some of your text. However, if that style comes with the program, nothing happens when you attempt to delete it.

Let's delete a style currently applied to some of your text by following these simple steps:

  • Select Format from the Menu bar
  • Select Style
  • Choose All from the List drop-down to view all the styles available
  • Highlight the style from the styles list that you want to delete
  • Select Delete
  • Choose Apply
  • Select Close

Congratulations! You have just completed all the lessons in Lesson 5 and should now be able to apply, create, modify, and delete a style.
