Apply a Style from the Style dialog box

You can apply a style from the Style dialog box. You may find this particularly helpful because it allows you to choose from a larger selection of styles than are available from the Styles drop-down menu and allows you to preview a style before you actually apply it to your document.

Let's apply a style using the Style dialog box by following these easy steps:

  • Place your cursor within the paragraph you want to add a particular style
  • Select Format from the Menu bar
  • Select Style
  • Choose All from the List drop-down menu to view all styles available
  • Preview several styles by highlighting the name of the style
  • Select the style you want to apply to the paragraph
  • Select Apply
  • Select Close

You have just completed this section of Lesson 5 and now know how to apply a style from the Style dialog box. If you'd like to learn how to create a new style, select Next.
