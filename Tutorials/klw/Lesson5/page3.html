Create a New Style

You may find that the styles in the Normal template do not have the characteristics you need for a particular document. AbiWord allows you to create a new style unique to your writing requirements.

Let's create a new style and apply it to some text in your document by following these simple steps:

  • Select Format from the Menu bar
  • Select Style

    The Styles dialog box appears.

  • Select New

    The New Style dialog box appears.

  • Type a name for the new style in the Style Name field
  • Select Paragraph or Character from the Style Type drop-down
  • Select the Paragraph at the bottom of the window to choose the paragraph elements that you would like to format for the style
  • Make alignment, indentation, and spacing changes in the Paragraph dialog box
  • Select OK to set the style
  • Select OK to close the New Style dialog box
  • Choose All from the List drop-down to view all the styles available and to make sure that your new style has been added
  • Select the new style you created
  • Select Apply
  • Select Close

You have just completed this section of Lesson 5 and now know how to create a new style. If you'd like to learn how to modify or rename style, select Next. If you'd like to go to a different section within this lesson or to a new lesson, choose from one of the options in the Navigation bar.
