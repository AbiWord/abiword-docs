Modify a Style

You can quickly change an existing file by redefining it in the Styles dialog. For example, if you set the main headings in your document to 14 pt Arial, flush left, and bold, and you decide you want to change your headings to 16 pt, Arial Narrow, and centered, you don't have to reformat every main heading in your document. Instead, just change the properties of that style.

Let's change an existing style by following these few easy steps:

  • Select Format from the Menu bar
  • Select Style
  • Choose All from the List drop-down to view all the styles available
  • Highlight the style from the Style list that you want to modify
  • Select Modify

    The Modify Styles dialog box appears.

  • Select the Paragraph button at the bottom of the window to choose the paragraph elements that you would like to modify
  • Make alignment, indentation, and spacing changes in the Paragraph dialog box
  • Select OK to set the style
  • Select OK to close the Modify Styles dialog box
  • Select Apply
  • Select Close

You have just completed this section of Lesson 5 and can now modify an existing style. If you'd like to learn how to delete a style, select Next. If you'd like to go to a different section within this lesson or to a new lesson, choose from one of the options in the Navigation bar.
