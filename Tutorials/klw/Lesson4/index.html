Overview

This section of the AbiWord tutorial contains quick and easy instructions for enhancing your documents such as inserting pictures, symbols, page numbers, headers, footers, and creating lists.

Objectives

At the end of this section of the tutorial, you will know how to perform the following functions in AbiWord:

  • Insert a picture
  • Insert a symbol
  • Insert page number
  • Insert header
  • Insert footer
  • Create a list

If you would like to move forward and learn how to insert a picture, select Next. If you'd like to go to a different section within this lesson or to a new lesson, choose one of the options in the Navigation bar.
