Create Lists

AbiWord allows you to create various kinds of lists within your documents. You can create a list using bullets or numbers to show items in a series. Bullets place a symbol at the beginning of each list item, and use the same symbol for each item, while numbered lists mark each item with a number or letter, and move upward numerically or alphabetically as the list progresses.

Create a Bullet list

Let's begin by creating a bullet list by following these few simple instructions:

  • Select Format from the Menu bar
  • Select Bullets and Numbering

    The Lists dialog box appears.

  • Select Bullet from the Type drop-down arrow
  • Select the bullet style you prefer from the Style drop-down arrow

    Note: You can also customize a bulleted list by changing some of the options in the Customized List box.

  • Select Apply
  • Select Close

Create a Numbered list

Now let's create a numbered listed by following these few basic steps:

  • Select Format from the top Menu bar
  • Select Bullets and Numbering

    The Lists dialog box appears.

  • Select Numbered from the Type drop-down
  • Select the number style from the Style drop-down

    Note: You can choose from a variety of number styles e.g., alpha, numeric, Hebrew, Arabic, or Roman Numerals. You can also customize a numbered list by changing some of the options in the Customized List box.

  • Select Apply
  • Select Close
