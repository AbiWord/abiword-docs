Insert Headers and Footers

Typically, you use headers and footers in printed documents. You can create headers and footers that include text or graphics — for example, page numbers, the date, a company logo, the document's title or file name, or the author's name. A header prints in the top margin; a footer prints in the bottom margin.

Basically, you insert headers and footers in the same manner. For the purposes of this tutorial, let's insert a header into your document by following these few simple instructions:

  • Select the Edit Header button from the Extra toolbar
  • Type some text within the Header box
  • Point your mouse arrow to an area located outside of the text box
  • Press the left mouse button on any white space to remove the box, and type elsewhere

Note: You can also insert a Header into your document by doing the following:

  • Select Edit from the Menu bar
  • Select Edit Header
  • Type some text within the Header box
  • Point your mouse arrow to an area located outside of the text box
  • Press the left mouse button on any white space to remove the box, and type elsewhere

Format Headers/Footers

You can use the same header and footer throughout a document or change the header and footer for part of the document by using the Format Headers/Footers feature in AbiWord. For example, use a unique header or footer on the first page, or leave the header or footer off the first page. You can also use different headers and footers on odd and even pages or for part of a document.

Let's format the headers and footers in your document by following these few simple instructions:

  • Select Format from the Menu bar
  • Select Format Headers/Footers
  • Select one or more of the options that best suits where you would like your Headers and Footers placed
  • Select OK

You have just completed this section of Lesson 4 and can now insert and format headers and footers. If you would like to move forward and learn how to insert a list, select Next.
